JavaScript: Conhecendo o Browser e padrões de projeto

document.querySelector

API do DOM, que permite buscar um elemento do DOM, usando o seletor CSS.

document.createElement

Para criar elemento no DOM (que ainda não faz parte dele).

forEach

Uma maneira de iterar o array sem necessidade de um loop for.

Iterar = fazer ou dizer novamente; repetir, reiterar.

textContent

O conteúdo de texto entre uma tag.

appendChild

Adiciona um filho.

event.preventDefault

Evita que a página seja recarregada.
